# Umlaut Converter

A TypeScript project for converting replacement letters into umlauts and generating SQL queries.

## Setup

1. Install dependencies:
    ```bash
    npm install
    ```

2. Configure the database connection in `src/config/dbConfig.ts`.

3. Build the project:
    ```bash
    npm run build
    ```

4. Run the project:
    ```bash
    npm start
    ```

5. Run tests:
    ```bash
    npm test
    ```
